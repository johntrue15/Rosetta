Step 4 — Install the Rosetta Edge Watchdog

Your facility has been approved and the branch is ready. Now install the watchdog on the machine where your scanner writes data.

1. Clone and install
git clone https://github.com/johntrue15/Rosetta.git cd Rosetta/edge pip install -e .
2. Create your config file

Click below to download a config.yml pre-filled with your facility details, or copy the contents and save manually.

3. Set your GitHub token
export ROSETTA_GITHUB_TOKEN="ghp_YourPersonalAccessTokenHere"

Create a Personal Access Token with repo scope.

4. Run the watchdog
rosetta-watchdog -c config.yml
