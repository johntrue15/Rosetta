How it works

  1. Drop your CT scanner metadata files above (or click to browse).
  2. Each file is committed to data/ in the repository.
  3. GitHub Actions parses it, moves the original to data/completed/, and writes JSON to data/parsed/.
  4. Results are aggregated into data/metadata.json and data/metadata.csv.
