How it works

  1. Drop your metadata files above (or click to browse).
  2. Each file is committed to the data/ directory in the repository.
  3. GitHub Actions automatically parses, aggregates, and exports your data.
  4. Results appear in data/parsed/, data/metadata.json, and data/metadata.csv.
